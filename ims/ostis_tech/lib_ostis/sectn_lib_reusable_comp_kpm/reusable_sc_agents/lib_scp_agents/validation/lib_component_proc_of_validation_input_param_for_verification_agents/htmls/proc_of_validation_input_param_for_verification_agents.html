Задачей scp-программы проверки входных параметров sc-агентов верирфикации базы знаний является проверки входных параметров sc-агентов верирфикации базы знаний на принадлежность разделу, структуре либо предметной области. Первым входным параметром данной scp-программы является знак действия. Вторым входным параметром данной scp-программы является аргумент действия. Третий параметр - возвращаемый результат выполнения (1 - ошибок нет, 0 - ошибки присутствуют).
